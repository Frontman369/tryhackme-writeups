# Windows PowerShell - TryHackMe Writeup

##  Overview
This room introduced PowerShell, a powerful command-line and scripting environment used in Windows systems. It covered the fundamentals of navigating the system, working with data, and using PowerShell for real-world cybersecurity tasks.



## What I Learned

- Navigating the filesystem using PowerShell  
- Working with files and directories  
- Using piping to pass output between commands  
- Filtering and sorting data efficiently  
- Retrieving system and network information  
- Performing basic system analysis  
- Writing and understanding simple PowerShell scripts  

---

## Practical Exercise

- Investigated a simulated system to identify an intruder  
- Collected important information such as:
  - User-related data  
  - Hash values  
  - Running services  
- Used PowerShell commands to analyze system activity and locate relevant evidence  

---

## Challenges

- Understanding the structure of PowerShell commands initially  
- Getting comfortable with piping and filtering data  
- Interpreting system information effectively  

---

## Key Takeaways

- PowerShell is a powerful tool for both system administration and security analysis  
- Piping and filtering make it efficient for handling large amounts of data  
- It plays a significant role in both offensive and defensive cybersecurity tasks  
- Scripting allows automation of repetitive tasks  

---

## Reflection

This room strengthened my understanding of Windows command-line environments beyond traditional CMD. The hands-on investigation made it more practical, showing how PowerShell can be used to analyze systems and identify suspicious activity.

I am still getting familiar with its full capabilities, but with more practice, I expect to become more efficient in using it for both system management and security-related tasks.
