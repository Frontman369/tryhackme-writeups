# Networking Core Protocols - TryHackMe Writeup

## Overview
This room introduced several core TCP/IP protocols that power everyday internet communication. It focused on how services such as web browsing, file transfers, domain resolution, and email operate behind graphical interfaces.

---

## Objectives
- Understanding the purpose of common networking protocols  
- Learning how domain names are resolved  
- Exploring how web, file transfer, and email communication function  
- Gaining insights into how protocols operate beneath GUI-based applications  

---

## Protocols Covered
- WHOIS  
- DNS  
- HTTP  
- FTP  
- SMTP  
- POP3  
- IMAP  

---

## Key Concepts

### DNS (Domain Name System)
DNS translates human-readable domain names into IP addresses, allowing systems to locate servers across networks.

---

### HTTP (Hypertext Transfer Protocol)
HTTP is used for communication between web browsers and web servers to deliver websites and web content.

---

### FTP (File Transfer Protocol)
FTP enables file transfers between systems across a network.

---

### SMTP, POP3 & IMAP
These protocols are responsible for email communication:

- **SMTP** → sending emails  
- **POP3** → downloading emails locally  
- **IMAP** → synchronizing emails across devices  

---

### WHOIS
WHOIS provides registration and ownership information related to domain names.

## Key Takeaways
- Modern applications rely heavily on underlying network protocols  
- DNS is fundamental for navigating the internet  
- Web, file transfer, and email communication each depend on specialized protocols  
- Many internet services function through processes users rarely see directly
