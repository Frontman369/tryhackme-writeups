# Become a Hacker - TryHackMe Writeup

## 🧠 Overview
In this room, I learned about the role of ethical hackers, became familiar with key terminology and hacking methodologies, and gained hands-on experience using real security tools used by professional red teamers to help secure a web application.

---

## 🛠️ Tools Used
- Gobuster → directory and file brute forcing
- Hydra → login brute force tool

---

## 🔍 What I Did

- Used Gobuster to discover hidden directories on a web application (directory enumeration)
- Used Hydra to attempt login brute force on a service

This gave me my first exposure to real-world enumeration and password attacks.

---

## ⚠️ Challenges

- I am not yet fully familiar with how Gobuster and Hydra work in depth
- Understanding when and how to use each tool effectively is something I need to improve

---

## 💡 Key Takeaways

- Enumeration is a critical first step in penetration testing
- Weak credentials can be exploited using brute force techniques
- Tools like Gobuster and Hydra are widely used in real-world scenarios

---

## 🚀 Reflection

This was my first hands-on pentesting experience. While I am still not very comfortable with these tools, continued practice will help me improve. This is just the beginning of my journey into cybersecurity and offensive security.
