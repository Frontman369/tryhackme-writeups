***Open Systems Interconnection*** model taught me some basics on how the internet actually works. 7 layers. Each one, acting on solving its own part. 

Imagine, you want to get dressed up. You will need clothes of all forms to cover your body. The same is the internet. We only think of the internet as a solo entity.
The reality is, it's made up of components, starting from Physical, all the way to the application.

This room made me question about what I thought I knew about how internet works, compared to how it actually does.

## Layers of OSI Model

1. Physical – Handles raw data transmission (cables, signals)
2. Data Link – MAC addresses, frames
3. Network – IP addressing and routing
4. Transport – TCP/UDP, reliable delivery
5. Session – Manages sessions
6. Presentation – Data formatting, encryption
7. Application – User interaction (HTTP, FTP)

## 📌 Real-World Example

When I visit a website:

- Application → HTTP request is made
- Presentation → Data is formatted (e.g., encryption/SSL)
- Session → Session is established between client and server
- Transport → TCP ensures reliable delivery
- Network → IP routes the packet
- Data Link → Frames are sent within local network
- Physical → Bits travel through cables

This helped me understand how each layer works together.
